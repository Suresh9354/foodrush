# FoodRush

**FoodRush** is an Online Food Delivery Web Application built using Java Full Stack technologies.

## Tech Stack
- **Backend**: Java 17, Jakarta Servlets, JSP, JDBC
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Application Server**: Apache Tomcat 10.1
- **Build Tool**: Maven

## Architecture
- Model-View-Controller (MVC) architecture pattern.
- Data Access Object (DAO) pattern for database separation.
- Transaction management for atomic database operations.
- BCrypt secure password hashing.
